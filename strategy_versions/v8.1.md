# V8.1 — Structural OB-Bottom Stop

发布日期: 2026-05-07

## 概述

V8.1在V8基础上将止损位从"影线最低点"改为"OB结构底部"，减少流动性猎杀。

## 入场条件

### OB建仓（位移K线检测）
- 核心时段(7-21h): body_pct > 0.55, vol_ratio > 0.9
- 扩展时段(22-6h): body_pct > 0.50, vol_ratio > 0.8
- 做多OB: 收盘 > prior_high_30m + 0.10×ATR + 阳线 + 前一根阴线
- 做空OB: 收盘 < prior_low_30m - 0.10×ATR + 阴线 + 前一根阳线
- OB区域 = 前一根K线的 [open, close]
- 动态TTL: 强OB(body>0.8+vol>1.5)=15根, 弱OB(body<0.7)=8根, 默认12根

### 回踩入场
- 5m K线low触及OB区域: `OB.bottom <= low[bar] <= OB.top`（做多）
- 方向确认: `close > open`（阳线确认做多）
- 入场价: `(OB.top + OB.bottom) / 2 + spread`

### 风控过滤
- `spread×1.5 < risk < ATR×3`
- 逆势过滤: 1H趋势反向 且 risk/ATR >= 1.5 → 跳过
- 夜盘19-21h: vol>1.3 + body>0.50 + risk/ATR<1.0
- 扩展22-6h: vol>1.5 + body>0.60 + risk/ATR<0.8 + 顺势

## 止损

**V8.1核心改动**: `stop = close[bar-1] - 0.15×ATR` (OB结构底部外扩)

- 之前V8: `stop = low[bar-1] - 0.08×ATR` (影线最低点外扩)
- 原因: OB底部=close[bar-1]是真正的结构支撑，low[bar-1]是影线噪音，容易被流动性猎杀

## 止盈/追踪

多级追踪止损:
- 0.5R → 锁0.1R（V7.8二推不破保护）
- 0.7R → 锁0.3R
- 1.2R → 锁0.6R
- 1.8R → 锁1.0R
- 2.5R → 锁1.5R
- 4.0R+ → 锁60%峰值

自适应动态止盈:
- ≥8R回撤18%
- ≥4R回撤22%
- ≥2.5R回撤28%
- 其他回撤32%

时间止盈: 80根K线后，浮盈>=0.5R即止盈
