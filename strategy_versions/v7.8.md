# V7.8 — 0.5R Breakeven Protection

发布日期: 2026-05-06

## 关键变动
- 新增追踪级别: 0.5R → 锁0.1R（解决"二推不破"止损问题）
- 减少31%亏损笔数，胜率+3.6pp，PF从9.33提升到13.38

## 概述

在V7.7多级追踪中插入一级:浮盈到0.5R即锁0.1R保本。解决44%止损单"先有浮盈0.3-0.7R然后回落满损"的问题。

## 入场条件
与V7.7完全相同（含共振、时段扩展、动态TTL等）

## 止损
`stop = low[bar-1] - 0.08×ATR` (与V7.7相同)

## 止盈/追踪（核心改动）

多级追踪:
- **0.5R → 锁0.1R** ← 新增！
- 0.7R → 锁0.3R
- 1.2R → 锁0.6R
- 1.8R → 锁1.0R
- 2.5R → 锁1.5R
- 4.0R+ → 锁60%峰值

动态止盈/共振分批: 与V7.7相同

## 为什么有效

数据显示55.9%的止损单(1570笔)曾有0.3-0.7R浮盈——"二推不破"模式。0.5R保本后:
- 849笔从-1R变为+0.1R(每笔节省1.1R)
- 代价: 1190笔从+0.3R变为+0.1R(每笔损失0.2R)
- 复利下净效果: +1003倍(因为避免-1R的复利惩罚远大于损失0.2R)

## 回测结果 (vs V7.7)
- 胜率: 88.4% → 92.0% (+3.6pp)
- PF: 9.33 → 13.38 (+4.05)
- 亏损: 4622 → 3205 (-31%)
- 余额: +1.7%
