# V8.3 — Three-Level Limit + Freshness/Continuation Boost

发布日期: 2026-05-07

## 关键变动 (vs V8.2)
- 三级入场: 从50%中点+50%深入 → 33%中点+33%下2/3+33%OB底部
- 新鲜度加仓: OB age≤3根时 pos_mult×1.5
- 位移延续加仓: 位移K线后下一根同方向时 pos_mult×1.3
- 两个加仓可叠加: 新鲜+延续 = ×1.95
- 余额: +11% vs V8.2

## 概述

在V8.2限价分批入场基础上,增加第三层入场位(OB底部)+两个仓位boost因子。新鲜OB结构更可靠(age≤3胜率最高),位移后续K线同方向证明动能充足,两者都值得加仓。

## 入场条件

### OB建仓 (与V8.2相同)
- 核心时段(7-21h): body>0.55, vol>0.9
- 扩展时段(22-6h): body>0.50, vol>0.8
- 动态TTL: 强OB=15根, 弱OB=8根, 默认12根
- 额外记录: next_same = 位移后下一根K线是否同方向

### 入场方式 (三级限价)

**不等5m收盘确认,OB识别后立即设三层限价:**

1. **中点限价 (33%仓位)**: `(OB.top + OB.bottom) / 2`
2. **下2/3限价 (33%仓位)**: `OB.bottom + (OB.top - OB.bottom) / 3`
3. **OB底部限价 (33%仓位)**: `OB.bottom + 0.02 × (OB.top - OB.bottom)`

触发: 5m K线low触及对应价位即入场(做多)。深入位和底部位不一定填充。

### 仓位Boost

基础仓位由V8多层分级决定(super_high 5x, high 3x...standard 1x)。

在此基础上叠加:
- **新鲜度**: `age <= 3` → ×1.5 (OB刚创建3根内,结构最新鲜)
- **延续确认**: `next_same = True` → ×1.3 (位移后下根K线方向一致)
- **叠加**: 满足两者 = ×1.95

最终每笔仓位 = `base_mult × fresh_mult × cont_mult / 3` (除以3因为三级分仓)

### 风控过滤 (与V8.2相同)
- `spread×1.5 < risk < ATR×3`
- 逆势过滤: 1H趋势反向且risk/ATR>=1.5 → 跳过
- 夜盘19-21h: vol>1.3 + body>0.50 + risk/ATR<1.0
- 扩展22-6h: 需通过质量分级(ext_L1或ext_L2)

## 止损

`stop = OB.bottom - 0.15×ATR` (V8.1结构止损,不变)

## 止盈/追踪 (与V8.2相同)

多级追踪:
- 0.5R → 锁0.1R
- 0.7R → 锁0.3R
- 1.2R → 锁0.6R
- 1.8R → 锁1.0R
- 2.5R → 锁1.5R
- 4.0R+ → 锁60%峰值

自适应动态止盈: ≥8R→18%, ≥4R→22%, ≥2.5R→28%, 其他32%

高质量/共振单分批: mult≥2时1.5R出50%,剩余宽追踪。

时间止盈: 80根后浮盈>=0.5R

## 冷却
- 同方向单笔亏损: 冷却6根
- 同方向连续亏损: 冷却12根
