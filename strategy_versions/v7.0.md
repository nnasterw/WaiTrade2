## 入场条件

### OB建仓 (位移K线)
- 时段: 7-16点
- body_pct > 0.60
- vol_ratio > 1.0
- 做多: 收盘 > prior_high_30m + 0.10×ATR + 阳线 + 前一根阴线
- 做空: 收盘 < prior_low_30m - 0.10×ATR + 阴线 + 前一根阳线
- OB区域 = 前一根K线的 [open, close]
- TTL = 9根5m K线

### 回踩入场
- 5m K线low触及OB区域: `OB.bottom <= low[bar] <= OB.top`
- 方向确认: `close > open` (阳线确认做多)
- 入场价: `(OB.top + OB.bottom) / 2 + spread`
- 风控: `spread×1.5 < risk < ATR×3`

## 止损
`stop = low[bar-1] - 0.08×ATR` (位移前一根K线影线最低点外扩)
