# v11j1 — BTC M5 专用候选

日期: 2026-05-21

## 定位

`v11j1` 由 `v11_btc_m5_r21` 固化而来，是 BTCUSDm M5 专用候选。核心思想是保留高质量 OB 反弹入场，只对日志验证过的右尾结构加权：

- H1 净推进分层保护反向动能。
- `confirm_pos > -0.60` 的 OB 边缘浅确认硬降权。
- 全局风险降到 `2.7%`。
- `risk >= 300` 的大结构集中放大到 `2.4x`。

## 核心参数

```yaml
v11j1:
  version: V11J1
  period: M5
  risk_percent: 2.7
  enable_htf_net_push_filter: true
  htf_net_push_tf: 60
  htf_net_push_bars: 3
  htf_net_push_min_atr: 0.35
  htf_net_push_aligned_mult: 1.15
  htf_net_push_neutral_mult: 1.0
  htf_net_push_counter_mult: 0.45
  shallow_confirm_pos_min: -0.60
  shallow_confirm_pos_mult: 0.45
  large_risk_min: 300.0
  large_risk_mult: 2.40
  max_pos_mult: 150.0
  max_lot_size: 3.20
```
