# V8.4 — Tighter Stop + Large OB Deep-Only + Trailing Acceleration + Tight TP

发布日期: 2026-05-07

## 关键变动 (vs V8.3)
- 止损收紧: 0.15ATR → 0.10ATR (减小risk,提高R)
- 大OB(>0.5ATR)跳过中点入场,只入深入+底部(更好价格)
- 追踪加速: 单根K线涨>=0.8R时提前收紧锁定
- 强位移收紧止盈: disp_strength>2时动态止盈回撤×0.7
- 余额: +21% vs V8.3, +520% vs V7.9

## 概述

在V8.3(三级限价+boost)基础上,通过四项精准微调进一步提升复利效率:
1. 更紧止损 → risk更小 → 每笔R更高 + 自动淘汰弱OB
2. 大OB不入中点 → 只在深入位成交,入场更优
3. 快利单追踪加速 → 及时锁住快速利润
4. 强位移单收紧回撤 → 大利润不回吐

## 入场条件

### OB建仓 (与V8.3相同)
- 核心时段(7-21h): body>0.55, vol>0.9
- 扩展时段(22-6h): body>0.50, vol>0.8
- 动态TTL: 强OB=15, 弱OB=8, 默认12

### 入场方式

**限价单, 不等确认:**

正常OB(size≤0.5ATR): 三级入场
- 33% OB中点
- 33% OB下2/3
- 33% OB底部(+0.02空间)

大OB(size>0.5ATR): 跳过中点
- 50% OB下2/3
- 50% OB底部
- 中点不入(价格太差)
- fallback: 如果深入和底部都未填充,以中点1/3仓位入场

### 仓位Boost (与V8.3相同)
- 新鲜度: age≤3 → ×1.5
- 位移延续: next_same → ×1.3
- 叠加: ×1.95
- 基础倍数: V8多层分级(5x/3x/2.5x/2x/1x)

### 风控过滤 (与V8.3相同)
- spread×1.5 < risk < ATR×3
- 逆势+大止损过滤
- 夜盘/扩展时段过滤

## 止损 (V8.4核心改动)

`stop = OB.bottom - 0.10×ATR` (从V8.3的0.15收紧到0.10)

原因: 之前buffer sweep显示0.10在复利下优于0.15——更紧止损让risk更小,虽然触发率稍高,但每次止损代价更轻。

## 止盈/追踪 (V8.4改进)

### 基础多级追踪 (与V8.3相同)
- 0.5R→锁0.1R, 0.7R→锁0.3R, 1.2R→锁0.6R, 1.8R→锁1R, 2.5R→锁1.5R, 4R+→锁60%

### V8.4追踪加速 (P)
- 当单根K线max_r增加>=0.8R时(快速获利):
  - 如果max_r>=2.5R: 锁定提升到max(lock, 1.8R)
  - 如果max_r>=1.2R: 锁定提升到max(lock, 0.8R)
- 作用: 快利单不给回吐机会

### V8.4强位移收紧止盈 (K)
- 当OB的位移强度(disp_strength = body_pct×vol_ratio) > 2.0时:
  - 动态止盈回撤阈值 × 0.7
  - 例: 原本32%回撤 → 22.4%回撤
- 作用: 强位移OB走出的利润更可靠,不应容忍过多回吐

### 高质量/共振分批 (与V8.3相同)
- mult>=2时: 1.5R出50%, 剩余宽追踪

时间止盈: 80根后浮盈>=0.5R
