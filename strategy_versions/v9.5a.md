# V9.5a — 1分钟周期 OB 策略

## 回测结果 (12品种×120天, 严谨成本模型)

- 净R: +52,392 (strict cost) / +54,776 (raw)
- 胜率: 66.5%
- 盈亏比: 11.49
- 日/品: 7.2单
- vs V9.5(5m): +994%
- vs V9.5(3m): +400%

## 核心配方 (与V9.5相同逻辑, 1m周期)

| 参数 | 值 |
|------|-----|
| K线周期 | **1分钟** |
| bounce | 30% |
| post_confirm | 30秒不回落 |
| 二推加仓 | +2x (总3x) |
| OB过滤 | >3×spread |
| risk/spread过滤 | ≥5 (严谨成本) |
| 1H OB boost | 3x |
| ds动能加权 | min(ds×1.5, 2.5) |
| 保本 | 0.5R→锁0.1R |
| trailing | 1.5R锁0.3R, 3R+锁peak×0.65 |
| timeout | 90min |
| 并发 | 5 |
| 冷却 | 0 |

## 为什么1m优于5m

1. 1m OB 更多更精确 — 信号量 8757 vs 1800 (4.9x)
2. 1m SL 更紧 — avg_loss 0.67R vs 1.09R
3. post30s 在1m上同样有效过滤噪音 — WR 67% vs 48%
4. 严谨成本(full spread + SL滑点1×spread + min risk/spread≥5)只降低4%

## 严谨成本验证

成本模型: full_spread_cost + sl_slippage=1×spread + min_risk_spread_ratio=5

| mid | 8757 | 66.0% | 12.24 | +52628 | -4% |

## Live要求

- 轮询间隔: ≤1秒 (1m K线级别)
- bars: 5000 (≈3.5天历史)
- minutes_per_bar: 1
- 适合品种: BTC/ETH (spread极小), XAU (spread适中)
- 慎用: 外汇高spread品种需确认 risk/spread≥5
