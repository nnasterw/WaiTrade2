# V8.0 — Multi-Level 24h OB Strategy

发布日期: 2026-05-06

## 关键变动
- 信号质量多层分级仓位: super_high 5x, high 3x, high_ext 2.5x, medium 2x, medium_B 2x, standard 1x
- 新增medium_B层: 逆势但强确认(vol>1.5+body>0.7)也给2x
- 新增high_ext层: vol>1.3+body>0.55+risk<0.9+顺势 → 2.5x
- 扩展时段分两级: ext_L1(严格)3x, ext_L2(放宽)2x
- 日均从18.9提升到21.7笔

## 概述

基于V7.9，将信号按多维度质量评分分为8个层级，每个层级对应不同仓位倍数。高质量信号获得更多资源，低质量信号仍保留但减仓。

## 入场条件

### OB建仓
- 核心时段(7-21h): body>0.55, vol>0.9
- 扩展时段(22-6h): body>0.50, vol>0.8 (放宽以适配ext_L2)
- 动态TTL: 强OB=15根, 弱OB=8根, 默认12根

### 回踩入场
- low触及OB + 阳线确认 + 风控检查
- 入场价: OB中点 + spread
