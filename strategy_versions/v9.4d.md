## 核心配方

| 参数 | 值 | 说明 |

| bounce阈值 | 30% | OB高度的30%反弹确认 |
| 二推窗口 | 90min | 两次触及间隔最大90分钟 |
| timeout | 90min | bounce确认等待时间(在window内) |

| 并发 | 5 | 二推验证后OB可信度高 |
| 冷却 | 0 | 无冷却 |

## "二推不破"逻辑

```
1. 价格第一次触及 OB 区域
2. 反弹离开 (bounce ≥ 30% × OB高度)
3. 不入场，标记 OB 为"已验证"
4. 在90min内，价格第二次触及同一 OB
5. 再次反弹 ≥ 30% → 入场
```

为什么有效：
- 第一次触及验证 OB 有需求/供给
- 第二次仍不破 = 该区域力量强，方向确认
- 过滤了 45% 的"死在入场"噪音信号

## 1H OB boost 逻辑

当 5m OB 信号的入场价恰好落在 1H 级别 OB 区域内时：
- 多周期共振 = "攻防一体位"
- 仓位 ×3（正常1x → 3x）
- 1H OB 活跃期: 24小时

## 保本覆盖点差

保本时 SL 设为 entry + 0.1R（而非 entry）：
- 确保平仓后扣除点差仍不亏
- 特别适合小本金/高spread品种

## 时间衰减止盈

| 持仓时间 | 行为 |
|----------|------|
| 0-50% | 正常trailing |
| 50-75% | DTP回撤降到25% (更快止盈) |
| 75%+ | 任何 >0.5R 即平仓 |
