# V9.4aa — V9.4a + 1H OB Boost 2x

发布日期: 2026-05-09

## 核心改动 (vs V9.4a)

在V9.4a基础上增加大周期OB仓位加倍:
- 当5m信号的入场价处于活跃1H OB区域内，且方向一致时，仓位×2
- 其他信号保持正常仓位
- 出场逻辑不变（5m trailing/DTP）

## 为什么 boost2x > 双仓策略

- boost2x: 同入口同出口，两份都用5m快止盈 → 赢时2倍
- 双仓: 5m快出+1H慢出 → 1H仓62%止损率侵蚀利润
- 根因: 5m OB反弹不保证1H趋势延续，大周期SL意义不大

## 1H OB 定义

```python
# 结构突破: close > rolling(6).max of prior 1H highs + 0.1×ATR
# OB区域: 突破前一根反向K的body
# 活跃期: OB形成后24小时内
# 方向一致: 5m long信号 + 1H long OB 重合
```
