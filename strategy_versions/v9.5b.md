## 执行流程 (live + 回测一致)

```
1. 1m K线上识别 OB (generate_ob_signals_v84)
   - OB高度 ≥ 3×spread
   - risk ≥ 5×spread

2. 价格触及 OB → bounce ≥ 30%×OB高度

3. 等30秒不回落 (post_confirm)
   - 30s内价格不跌回 planned_entry
   - 回落则放弃 (零损失)

4. 通过后挂限价单在 planned_entry
   - 等待回调成交 (最多10分钟)
   - 不回调 → 放弃 (零损失)

5. 成交后持仓管理:
   - 0.5R → 保本 (锁0.1R覆盖点差)
   - 1.5R → 锁0.3R
   - 3.0R → 锁 peak×0.65
   - 2.0R+回撤35% → 动态止盈
   - 12min后时间衰减止盈

6. [可选] 入场后90min内OB被二次验证 → 加仓+2x(总3x)
```

## 加权逻辑

- **1H OB boost**: 入场价在1H OB区域内 → pos_mult × 3
- **ds动能加权**: pos_mult × min(ds × 1.5, 2.5)
- **二推加仓**: 入场后OB被再次触及+反弹 → pnl × (1 + 2.0)
